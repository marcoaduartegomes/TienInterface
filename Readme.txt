Bem vindo ao sistema de cadastro de emoticons e emoções. Para começar é necessario que o site esteja rodando num servidor para execudar o codigo PHP, e um banco de dados com o nome tien instalado( na pasta BD tem o tien.sql, basta importar no seu banco).

/////////////////////////////////////Desenvolvimento/////////////////////////////////////////////

Os itens e funções do sistema serão descritos a baixo, e depois colocados no nome de cada pessoa responsavel pelo seu desenvolvimento.

1- Criação do menu lateral, chamado de sideBar, onde você pode escolher para que aba vai se redirecionar;
2- Pagina inicial, para dar boas vindas ao usuario;
3- Pagina do emoticon, criada para cadastrar um emoticon, dado que sua emoção ja existe no sistema. Criação do modal e da tabela;
4- Pagina parecida com a do emoticon, mas voltada para cadastrar uma emoção;
5- Pagina Help, criada para ter uma informação basica do que cada aba faz, redirecionando para a mesma;
6- Criação do banco de dados para armazenar as informações;
7- Conexão do banco de dados na parte do cadastro de emoticon, onde pode cadastrar e deletar;
8- Conexão do banco de dados na parte do cadastro de emoções, onde pode cadastrar e deletar;
9- Criação do desenho virtual num ambiente grafico web;
10- Geração automatica de desenhos virtuais, e seu armazenamento.
